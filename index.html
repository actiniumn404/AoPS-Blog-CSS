Green.css is a nice, soothing CSS that is basically a green version of the default AoPS CSS. You can find the release here, and the code here.

White on Black.css is a simple dark mode CSS. As the title suggests, the CSS mainly consists of white features on a black background, plus some additional "special effects". You can find the release here, and the code here.

Unity.css was my first CSS designed after a webpage or software. It was designed after the Unity Game Engine. I would like to thank @Bachsonata3 for suggesting this in a DM, and @Happycupcake for multiple code snippets. This CSS features the navigation bar and title as the toolbar at the top of the unity software, the archives as the file navigator at the left of the screen, the shouts as the console, and the rest of the sidebar as the custimization options at the bottom right of the screen. You can find the release here, and the code here.

AoPS Class Homework.css is based off of the homework page of an AoPS Classroom. Kudos to the pr0 CSS, AIME, Math, and Alcumus god @Zhaom for some code snippets and overall helping me with the code. You can find the release here, and the code here.

Zoom.css is a CSS that is based off of the Zoom Meeting Software produced by Zoom Video Communications Inc. This features a flexbox repersentation of the posts as people in a zoom meeting, the contributors and shouts as the participants and chat windows to the right side of the screen respectively, and the rest of the sidebar with special hover effects as the menu bar at the bottom of the screen. You can find the release here, and the code here.

Gmail.css is probably one of my most accurate CSS's to this date. As you might guess from the title, Gmail.css is based off of the email service "Gmail" created by Google Inc. It features the navigation bar to the left side, and the posts as emails to the right. You can find the release here, and the code here.

Khan Academy.css is a simple repersentation of the Khan Academy website. You can find the release here, and the code here.

Github.css is my most accurate CSS up to the date of which I am writing this. This CSS is an accurate repersentation of a GitHub repo (along with some components off of a GitHub Issue and a GitHub wiki page) with both light and dark modes. You can find the release here, and the code here.
